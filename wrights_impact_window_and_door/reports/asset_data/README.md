# Asset Data Reports

This directory contains Google Ads asset performance and configuration data for Performance Max and Display campaigns.

## 📄 Ad asset report.csv

### What This File Represents
**Asset Performance Data** - Individual performance metrics for all ad assets across campaigns

### Key Data Included
- **Asset Types**: Images, videos, headlines, descriptions, sitelink extensions
- **Performance Metrics**: Impressions, clicks, conversions, cost per asset
- **Asset Status**: Approval status and policy compliance indicators
- **Campaign Association**: Which campaigns each asset is serving in
- **Asset Quality**: Google Ads quality assessments and recommendations

### Primary Use Cases
- **Asset Optimization**: Identify top-performing vs. underperforming creative assets
- **Quality Assessment**: Review asset approval status and policy compliance
- **Creative Strategy**: Understand which asset types drive conversions
- **Budget Allocation**: Allocate spend toward high-performing asset combinations

### Key Columns to Analyze
- **Asset Type**: Image, video, text, sitelink categorization
- **Performance Metrics**: CTR, conversion rate, cost per conversion
- **Approval Status**: Approved, disapproved, or pending review status
- **Policy Topics**: Specific policy violations or compliance issues
- **Asset Group**: Performance Max asset group associations

### Strategic Insights Available
- **Creative Performance**: Which asset types generate highest ROI
- **Policy Compliance**: Assets causing campaign disapproval
- **Asset Group Effectiveness**: Performance Max creative combinations
- **Quality Trends**: Asset quality score patterns and improvements

## 📄 Asset groups report.csv

### What This File Represents
**Performance Max Asset Group Data** - Configuration and performance of asset group combinations

### Key Data Included
- **Asset Group Structure**: Which assets are combined in each group
- **Performance Metrics**: Impressions, clicks, conversions by asset group
- **Targeting Settings**: Audience signals and geographic targeting
- **Bid Performance**: Cost and conversion data at asset group level
- **Creative Mix**: Headlines, descriptions, images, videos in each group

### Primary Use Cases
- **Performance Max Optimization**: Optimize asset group combinations for better performance
- **Audience Targeting**: Review audience signals and their effectiveness
- **Creative Testing**: A/B test different asset combinations
- **Budget Optimization**: Allocate budget to highest-performing asset groups

### Key Columns to Analyze
- **Asset Group Name**: Campaign and asset group identification
- **Performance Data**: Impressions, clicks, conversions, cost
- **Audience Signals**: Demographic and interest-based targeting
- **Asset Composition**: Number and types of assets in each group
- **Status**: Asset group approval and serving status

### Strategic Insights Available
- **Creative Synergy**: Which asset combinations perform best together
- **Audience Resonance**: Which audience signals drive conversions
- **Geographic Performance**: Asset group effectiveness by location
- **Scale Opportunities**: High-performing groups for expansion

## Analysis Strategy for Asset Data

### Cross-Reference Analysis
1. **Asset vs. Asset Group Performance**: Compare individual asset performance within groups
2. **Campaign-Level Impact**: How asset performance affects overall campaign metrics
3. **Geographic Variations**: Asset effectiveness across different target locations
4. **Time-Based Trends**: Asset performance changes over time

### Optimization Opportunities
- **Asset Refresh**: Replace underperforming creative assets
- **Asset Group Restructuring**: Reorganize high-performing asset combinations
- **Audience Expansion**: Scale successful audience signals
- **Policy Resolution**: Fix disapproved assets causing campaign issues
