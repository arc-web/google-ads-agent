# FullTiltAutobody.com - Campaign Assets

This directory contains all campaign assets for FullTiltAutobody.com Google Ads campaigns. Assets are organized by type and campaign focus for easy management and Performance Max campaign implementation.

## 📁 Directory Structure

```
assets/
├── images/                    # Static campaign images
├── videos/                    # Video assets for campaigns
├── logos/                     # Brand and partner logos
├── before_after_photos/       # Repair result showcases
├── service_specific/          # Service-focused asset collections
└── README.md                 # This file
```

## 🎨 Asset Specifications

### Performance Max Campaign Requirements

#### Image Assets (Required)
- **Format**: JPG, PNG (max 5MB each)
- **Dimensions**: Multiple sizes required (minimum 300x300px)
- **Aspect Ratios**:
  - Square: 1:1 (1200x1200px recommended)
  - Landscape: 1.91:1 (1200x628px recommended)
  - Portrait: 4:5 (960x1200px recommended)
- **Quality**: High resolution, professional appearance
- **Quantity**: Minimum 3-5 images per asset group

#### Video Assets (Optional but Recommended)
- **Format**: MP4, MOV
- **Duration**: 6-15 seconds (ideal for auto play)
- **Resolution**: 1080p minimum (1920x1080px)
- **File Size**: Max 200MB
- **Content**: Engaging, professional repair demonstrations

#### Text Assets (Required)
- **Headlines**: 30 characters max, compelling calls-to-action
- **Descriptions**: 90 characters max, benefit-focused messaging
- **Business Name**: Official business name
- **Call-to-Action**: "Call Now", "Get Estimate", "Schedule Service"

## ⚖️ Legal & Compliance Considerations

### Image Usage Rights
- All images must be original or properly licensed
- Customer photos require signed release forms
- Vehicle photos must not show identifiable license plates
- Professional photography recommended for brand consistency

### Content Guidelines
- No misleading claims about repair times or costs
- Insurance partnership claims must be verifiable
- Emergency service availability must be accurate
- Warranty information must match actual policies
