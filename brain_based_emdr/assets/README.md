# BrainBasedEMDR Campaign Assets

This directory contains campaign assets for Google Ads campaigns, including images, videos, and other media files used in Performance Max campaigns.

## 📁 Asset Organization

```
assets/
├── images/                 # Campaign images and graphics
├── videos/                 # Video content for campaigns
├── logos/                  # Brand logos and variations
├── screenshots/            # Website screenshots for RSA campaigns
└── README.md              # This file
```

## 🎨 Asset Guidelines

### Image Assets
- **Format**: PNG, JPG, GIF (max 5MB each)
- **Dimensions**: Minimum 300x300px, aspect ratios: 1:1, 4:3, 16:9
- **Content**: Professional, clinical presentation
- **Compliance**: No PHI, trauma-triggering imagery, HIPAA compliant

### Video Assets
- **Format**: MP4, MOV (max 100MB each)
- **Length**: 6-15 seconds for PMAX, up to 30 seconds for YouTube
- **Content**: Educational, non-triggering, evidence-based focus
- **Audio**: Optional background music, clear narration

### Logo Assets
- **Primary Logo**: Horizontal and vertical variations
- **Background**: Transparent PNG preferred
- **Minimum Size**: 500x200px for horizontal, 200x500px for vertical

## 📋 Asset Requirements by Campaign

### Performance Max Campaigns
- **Minimum Assets**: 3 images, 1 logo, 1 video (recommended)
- **Image Types**: Hero images, lifestyle shots, testimonial graphics
- **Video Content**: Treatment process explanation, success stories (anonymized)

### Search Campaigns (RSA)
- **Logo**: Required for brand verification
- **Screenshots**: Website landing page screenshots
- **Business Images**: Office/clinic photos, clinician headshots

## 🚫 Content Restrictions

### Healthcare Compliance
- **No Medical Claims**: Cannot claim to cure or guarantee outcomes
- **Evidence-Based Only**: Use research-supported statements
- **No PHI**: Never include protected health information
- **Trigger Warning**: Avoid potentially triggering imagery

### Professional Standards
- **Clinical Presentation**: Maintain professional, therapeutic environment
- **Diversity Inclusion**: Represent diverse client demographics
- **Accessibility**: Ensure content is accessible to all users

## 📝 Asset Naming Convention

BrainBasedEMDR_Video_EMDR_Process_20251216.mp4

## 🔄 Asset Upload Process

1. **Prepare Assets**: Follow guidelines above
2. **Quality Check**: Ensure HIPAA compliance and professional standards
3. **Upload to Google Ads**: Use Google Ads Editor or interface
4. **Performance Monitoring**: Track asset performance and rotate as needed

## 📊 Asset Performance Tracking

Monitor these metrics for each asset:
- **Impression Share**: Percentage of impressions won
- **Click-Through Rate**: Clicks per impression
- **Conversion Rate**: Actions per click
- **Cost Efficiency**: Cost per conversion
