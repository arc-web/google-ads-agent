
# Wright's Impact Window and Door Performance Max Campaign Rollout Strategy

## Overview
This strategy implements Performance Max campaigns for Wright's Impact Window and Door focusing on Lee County and Broward County, Florida. focusing on Lee County and Broward County, Florida. The strategy emphasizes service-specific + regional targeting with city-level "near me" asset groups.

## Campaign Structure
- **wrights_lee_county_pmax**: Lee County focus (West Coast)
- **wrights_broward_county_pmax**: Broward County focus (East Coast)
- **Asset Groups**: Service + Regional + City-specific Near Me targeting
- **Format**: Performance Max (asset group based, not keyword-based)

### Objectives
- Set up campaign structure and basic targeting
- Create service + regional asset groups
- Establish geographic boundaries

### Implementation Steps
1. **Create wrights_lee_county_pmax campaign**
   - Target: Lee County, FL (county-level)
   - Asset groups: impact_windows_regional, impact_doors_regional, hurricane_protection_regional, energy_efficiency_regional, commercial_solutions_regional
   - Budget: $500/day initially

2. **Create wrights_broward_county_pmax campaign**
   - Target: Broward County, FL (county-level)
   - Asset groups: impact_windows_regional, impact_doors_regional, hurricane_protection_regional, energy_efficiency_regional, commercial_solutions_regional

## Technical Implementation Notes
### Asset Group Structure
- **Service + Regional**: County-wide service targeting
- **City Near Me**: ZIP code specific local targeting
- **No Generic Near Me**: Each city has dedicated asset group

### Geographic Targeting
- County-level for service groups
- City + ZIP code level for near me groups
- No radius-based targeting

### Keyword Strategy
- Service-specific terms for regional groups
- "near me" + city name for local groups
- No generic "services near me" - all specific

## Risk Mitigation
- Start with conservative budgets
- Monitor geographic performance closely
- Have rollback plan for underperforming campaigns
- Validate asset approval before full rollout

## Success Criteria
- Positive ROI within 30 days
- Geographic attribution accuracy
- Asset group performance differentiation
- Conversion tracking validation
