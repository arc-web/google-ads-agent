# Collab Med Spa Campaign Assets

This directory contains all creative assets for Collab Med Spa Google Ads campaigns.

## Directory Structure

```
assets/
├── images/          # Campaign images and graphics
├── videos/          # Video content and demonstrations
├── documents/       # Supporting documents and PDFs
└── README.md       # This file
```

## Asset Requirements

### Images
- **Format**: JPG, PNG (high quality)
- **Dimensions**: 1200x628 for carousel, 300x300 for square
- **File Size**: <5MB per image
- **Content**: Before/after photos, service demonstrations, trust badges

### Videos
- **Format**: MP4 (H.264 codec)
- **Duration**: 15-60 seconds
- **Resolution**: 1920x1080 (1080p)
- **Content**: Service process, treatment demonstrations, customer testimonials

### Documents
- **Format**: PDF preferred
- **Content**: Service brochures, certifications, guarantees
- **Usage**: Sitelink and document assets

## Creative Guidelines

### Brand Colors
- Primary: [To be determined from website]
- Secondary: [To be determined from website]
- Trust: Green for guarantees, Blue for professionalism

### Messaging Focus
- Professional medical spa services
- Licensed and certified practitioners
- Advanced aesthetic treatments
- Local expertise (Scottsdale/Phoenix area)
- Trust and reliability

### Prohibited Content
- Before photos only (without after comparison)
- Unlicensed imagery
- Competitor mentions
- Unverified claims
- Medical claims not supported by evidence

## File Naming Convention

Format: `ServiceType_AssetType_Descriptor_Number.Extension`

Examples:
- `Facial_BeforeAfter_Hydrafacial_001.jpg`
- `Filler_Treatment_LipFiller_001.jpg`
- `Certification_MedicalBoard_Licensed_001.png`
- `Video_Testimonial_CustomerStory_001.mp4`

## Asset Review Process

1. **Creation**: Assets created following brand guidelines
2. **Review**: Internal team review for brand compliance
3. **Testing**: Google Ads preview and approval testing
4. **Launch**: Assets uploaded and campaigns launched
5. **Monitoring**: Performance tracking and optimization
