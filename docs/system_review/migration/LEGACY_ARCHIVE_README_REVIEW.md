# Legacy Archive

This directory contains old client folders, old planning documents, and historical examples that should not drive new `Google_Ads_Agent` rebuilds.

## Active Process

Use:

- `docs/GOOGLE_ADS_AGENT_PROCESS.md`
- `docs/CLIENT_DIRECTORY_SCAFFOLDING.md`
- `templates/client_template/`
- `shared/rebuild/scaffold_client.py`

## Archive Contents

- `root_clients/`: old root-level client directories moved out of the active path.
- `legacy_plans/`: old planning docs that predate the current process.
- `client_examples/`: historical client-specific examples.

## Rule

Do not copy instructions from this archive into active work unless they are reviewed against the current `Google_Ads_Agent` process.
